# @tilt-to-edit/embed

`@tilt-to-edit/embed` is the browser-first script-tag adapter for `tilt-to-edit`.

It ships:

- an ESM bundle for direct imports
- an IIFE bundle for `<script src="...">` embeds
- declarative auto-mounting via `data-tilt-to-edit`
- a global `window.TiltToEdit` API for manual mounting and instance control
- Shadow DOM widgets and DOM custom events for host-page integrations

This package was introduced in `v0.3.0`.
