# Architecture Decision Records

This directory contains the implemented ADR set for `tilt-to-edit`.

- `0001` - Workspace and layering strategy
- `0002` - Primary sensor backend choice
- `0003` - Permission and capability diagnostics
- `0004` - Calibration and screen-orientation normalization
- `0005` - Edit-intent processing pipeline
- `0006` - Explicit commit interaction model
- `0007` - First-party React adapter
- `0008` - Simulation and testing strategy
- `0009` - Examples and GitHub Pages demo
- `0010` - Versioning and changelog policy
- `0011` - Script-tag embed package with Shadow DOM widgets and custom events
- `0012` - Component-scoped setting menus for direct adjustment and submenus
